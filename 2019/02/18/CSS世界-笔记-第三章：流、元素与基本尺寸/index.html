1. 元素的显示类型

通常来说，元素分为块级元素和内联元素两种

使用 display 属性可以指定了元素的显示类型

语法

display: block | inline | inline-block

块级元素(block)的特点：
1. 每个块级元素会从新的一行开始
2. 存在盒模型，可设置宽高和内外边距
3. 宽度不设置的情况下，是父容器的100%

内联元素(inline)的特点：
1. 与其他内联元素显示在一行上
2. 没有盒模型
3. 元素宽度就是它包含的文字或图片宽度，不可变

行内块元素(inline-block)的特点:
1. 与行内块元素, 内联元素显示在一行上
2. 存在盒模型

常见的块级元素：

  • div
  • p
  • ul
  • section

常见的内联元素：

  • a
  • img
  • span
  • button
  • input

width/height的作用细节

张老师在这一块讲的很杂糅, 总结了一下, 大概有这几点

  1. 宽度分离原则：CSS 中的 width 属性不与影响宽度的 padding/border 属性共存
    也就是说将宽度控制放在父容器, 子元素来实现padding和border

  2. box-sizing 会改变 width/height
    准确来说是这个属性box-sizing: content-box;, 也是默认属性
    使用了该属性：真实容器的宽会等于 = width + border*2 + padding*2
    最好使用box-sizing: border-box, 这样真实容器的宽度 = width

  3. height:100% 失效
    如果父容器的高度没有设置为默认值, height会失效

  4. 权重 min > max > !important
    min-width/height > max-width/height > !important
